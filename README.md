# 区块链项目运行与数据查看指南

本指南用于快速启动项目的
**区块链节点、数据库、后端、IPFS、前端以及BERT语义查重服务**，并演示如何查看链上存证记录与下载文件。

------------------------------------------------------------------------
# 配置.env文件：
在当前的 xiyuan_backend 目录下，新建一个名为 .env 的文件，然后把下面的内容复制进去：
# 数据库连接地址 (如果你用的是本地 MongoDB)
MONGODB_URI=mongodb://127.0.0.1:27017/xiyuan_db
# 智能合约交互所需的私钥 (填一个刚才跑 npx hardhat node 时终端打印出来的 0x 开头的 Private Key)
PRIVATE_KEY=0x...你的测试私钥...
# 服务器端口 (可选，日志显示默认是 3000)
PORT=3000

# 一、项目启动流程（WSL Ubuntu 终端）
一键启动版本：
# 赋予脚本执行权限（仅首次需要）
chmod +x start_all.sh
chmod +x setup.sh
# 一键启动所有服务
./setup.sh
./start_all.sh

手动多个终端：
需要 **开启多个终端并保持全部运行**。

## 1. 启动区块链节点（Hardhat）

在 **终端1** 执行：

``` bash
cd ~/xiyuan-202603/xiyuan-202603/xiyuan_smart_contract
npx hardhat node
```

------------------------------------------------------------------------

## 2. 启动 MongoDB 数据库

在 **终端2** 执行：

``` bash
sudo mongod --dbpath /var/lib/mongodb --bind_ip 127.0.0.1
```

------------------------------------------------------------------------

## 3. 启动后端 API 服务

在 **终端3** 执行：

``` bash
cd ~/xiyuan-202603/xiyuan-202603/xiyuan_backend
npm start
```

------------------------------------------------------------------------

## 4. 启动 IPFS 节点

在 **终端4** 执行：

``` bash
ipfs daemon
```

------------------------------------------------------------------------

## 5. 启动前端

在 **终端5** 执行：

``` bash
cd ~/xiyuan-202603/xiyuan-202603/xiyuan_frontend
npm run dev
```

------------------------------------------------------------------------

## 6. 启动 BERT 语义查重服务

在 **终端6** 执行：

``` bash
cd ~/xiyuan-202603/xiyuan-202603/xiyuan_backend/bert_service

# 第一次使用需要创建虚拟环境
python -m venv venv

# 激活虚拟环境
source venv/bin/activate

# 第一次需要安装依赖
pip install -r requirements.txt

# 启动服务
python bert_service.py
```

------------------------------------------------------------------------

## 7. 打开前端页面

浏览器访问：

    http://localhost:5173/

操作步骤：

1.  连接钱包
2.  进行确权
3.  提交确权信息

------------------------------------------------------------------------

# 二、查看链上存证记录

## 1. 确保项目已经全部启动

即完成 **第一部分的所有终端操作**。

------------------------------------------------------------------------

## 2. 打开 MongoDB Shell

新开终端执行：

``` bash
mongosh
```

------------------------------------------------------------------------

## 3. 查看所有数据库

``` bash
show dbs
```

------------------------------------------------------------------------

## 4. 切换到项目数据库

``` bash
use xiyuan_blockchain
```

------------------------------------------------------------------------

## 5. 查看数据库中的集合

``` bash
show collections
```

------------------------------------------------------------------------

## 6. 查询存证记录

``` bash
db.researches.find().pretty()
```

该命令会格式化输出所有存证数据。

------------------------------------------------------------------------

# 四、整体流程总结

启动顺序建议：

1️⃣ Hardhat区块链节点\
2️⃣ MongoDB数据库\
3️⃣ 后端API服务\
4️⃣ IPFS节点\
5️⃣ 前端服务\
6️⃣ BERT查重服务\
7️⃣ 浏览器访问前端

随后即可：

-   提交研究确权
-   将文件上传至 IPFS
-   在区块链记录存证
-   在 MongoDB 查询数据
-   通过 IPFS 下载文件
